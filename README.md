# Skills

A curated collection of AI prompts organized by category. Use these with Claude Code, Cursor, or any AI assistant to get better results on specific tasks.

## Structure

```
skills/
├── coding/     # Code review, debugging, refactoring, explaining code
├── writing/    # Emails, documentation, copywriting
├── analysis/   # Architecture reviews, data exploration, problem solving
└── creative/   # Brainstorming, ideation
```

## Quick Start

Pick a category, find a prompt that matches your task, and paste it into your AI assistant. Each prompt is self-contained and ready to use.

## Categories

### Coding
Prompts for analyzing, reviewing, and improving code.

- [Project Analysis](coding/project-analysis.md) — Explore and understand unfamiliar codebases

### Writing
Prompts for professional writing tasks — emails, docs, marketing copy.

### Analysis
Prompts for deep thinking, architecture decisions, and exploring complex topics.

### Creative
Prompts for ideation and creative brainstorming.

## Adding Your Own Prompts

1. Choose the most relevant category
2. Create a new `.md` file with a descriptive name
3. Keep it focused — one prompt per file
